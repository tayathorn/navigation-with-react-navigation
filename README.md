# Navigation with react-navigation

## Installation
- `git clone https://github.com/tayathorn/navigation-with-react-navigation.git`
- `cd navigation-with-react-navigation`
- `npm install` or `yarn`

## Running

- `react-native run-ios`
- `react-native run-android`
